Nos dê uma chance. Você vai amar!!

A maioria dos nossos usuários vem do Windows e nunca olham para trás.

Porque escolher o Nil Linux??

Facíl para usar, rapido e confortavel

OOBT

Tudo funciona "Out Of The Box", sem a necessidade de configurar nada ou instalar aplicativos extras. Está pronto para ir. Embora seja muito configurável, tudo é definido e ajustado bem por padrão e nenhuma pergunta desnecessária é feita.

KISS

É muito fácil de usar. Possui um desktop intuitivo e adota os princípios KISS. Qualquer um pode se sentir rapidamente em casa e usar o Nil Linux. A experiência do usuário, o fluxo de trabalho e o conforto são fundamentais.

Regra Inicial

É o seu computador, suas regras. Este é um princípio chave no Nil Linux. Não coletamos dados, não trabalhamos contra você. Você é o chefe. Seu sistema operacional foi projetado para fazer o que você quer sem atrapalhar.

Base Solida

Com atualizações de software centralizadas, instantâneos do sistema, um gerenciador de atualizações exclusivo e a robustez de sua arquitetura Linux, o Nil Linux requer muito pouca manutenção. Funciona, é seguro e não quebra.

Gratuito e de código aberto

É totalmente gratuito e quase todos os seus componentes são Open Source. O Nil Linux está no ombro de gigantes, é baseado no Debian e no Ubuntu.

Centrado na comunidade

Existe um vínculo estreito e um ótimo relacionamento entre a equipe de desenvolvimento e a comunidade. Nós nos comunicamos com nossos usuários e usamos constantemente seus comentários para melhorar o Nil Linux.

Possiveis perguntas

Divertido, útil e encantador.

Se você está dando seus primeiros passos com o Linux, não hesite em participar dos fóruns . Os usuários do Linux geralmente são receptivos, prestativos e ansiosos para compartilhar suas experiências.

É gratuito?

Sim, o Nil Linux é totalmente gratuito. Quase tudo no Nil Linux também é de código aberto

Vai funcionar no meu computador?

Sim, o Nil Linux funciona na maioria dos computadores. Ele também pode ser executado a partir de um pendrive ao vivo para garantir que tudo funcione bem sem precisar instalar nada.

Posso manter o windows?

Sim, você pode ter Windows e Nil Linux. Um menu pergunta qual usar ao iniciar o computador.

Será que vai atender às minhas necessidades?

Esperançosamente. O Nil Linux vem com o que a maioria das pessoas precisa, além de fácil acesso a mais de 60.000 pacotes de software e 7.800 jogos.
